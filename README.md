# Español SRS — PWA 설치 안내

스페인어 문맥 기반 간격 반복(SRS) 학습 앱입니다. 폰 홈 화면에 설치해 앱처럼 쓸 수 있고, 한번 열어두면 오프라인에서도 복습이 됩니다.

## 파일 구성

```
index.html        앱 본체 (모든 로직 포함)
manifest.json     앱 이름·아이콘·테마 정의
sw.js             서비스워커 (오프라인 캐싱)
icons/            앱 아이콘 (192, 512, maskable, apple-touch)
spanish_sample.csv  문장 추가용 샘플 CSV
```

## 중요: PWA는 HTTPS에서만 설치됩니다

파일을 그냥 더블클릭해서 열면(`file://`) 서비스워커와 "홈 화면에 설치"가 동작하지 않습니다. 아래 중 하나로 **웹에 올려야** 합니다. 전부 무료입니다.

### 방법 1. GitHub Pages (추천, 영구 무료)

1. GitHub 계정 생성 후 새 저장소(repository) 만들기
2. 이 폴더의 파일들(`index.html`, `manifest.json`, `sw.js`, `icons/`)을 업로드
3. 저장소 Settings → Pages → Source를 `main` 브랜치 / 루트로 설정
4. 몇 분 뒤 `https://사용자명.github.io/저장소명/` 주소가 생김
5. 그 주소를 폰 브라우저에서 열기

### 방법 2. Netlify Drop (가장 빠름, 계정도 선택)

1. https://app.netlify.com/drop 접속
2. 이 폴더를 통째로 드래그 앤 드롭
3. 즉시 `https://...netlify.app` 주소 발급 → 폰에서 열기

### 방법 3. 내 컴퓨터에서 로컬 테스트만

```bash
cd 이폴더
python3 -m http.server 8000
```
같은 와이파이의 폰 브라우저에서 `http://컴퓨터IP:8000` 접속.
(단 localhost가 아니면 일부 브라우저는 설치를 막을 수 있어, 실제 설치는 방법 1·2 권장)

## 폰에 설치하기

- **Android (Chrome):** 주소를 열면 하단/메뉴에 "앱 설치" 또는 "홈 화면에 추가"가 뜸. 앱 안의 "홈 화면에 설치하기" 버튼으로도 가능.
- **iPhone (Safari):** 공유 버튼(□↑) → "홈 화면에 추가". iOS는 반드시 Safari로 열어야 함.

설치하면 아이콘이 생기고, 전체화면(주소창 없이)으로 실행됩니다.

## 데이터 저장에 대해

학습 진도는 그 브라우저의 `localStorage`에 저장됩니다. 즉 같은 주소를 같은 폰·같은 브라우저로 열면 이어집니다. 기기를 바꾸거나 브라우저 데이터를 지우면 진도가 초기화되니, 중요한 단어는 CSV로 따로 보관해두는 걸 권합니다.

## 문장 추가

앱 홈의 "CSV로 문장 추가"에서 CSV 파일을 넣으면 됩니다. 형식은 첫 줄이 헤더:

```
es,blank,ko,word,hint
```

- `es` 스페인어 문장 (필수)
- `blank` 빈칸으로 만들 단어 — 문장에 그대로 포함돼야 함 (필수)
- `ko` 한국어 뜻 (필수)
- `word` 핵심 단어 뜻 (선택)
- `hint` 영어 힌트 (선택)

쉼표·따옴표가 든 문장은 큰따옴표로 감싸면 됩니다. 한글이 깨지면 CSV를 UTF-8로 저장하세요.
